Below are recent technical projects that I've worked on, both inside and outside of class.

2022

Homemade is a platformer and a visual novel game utilizing full-stack web development in React and HTML/CSS. Synopsis: a small child cooks meals for their mother, who is sick. Created with Yuri Choi and Neil Malur for web.lab (6.9620), a website development class and competition.

2021

Snek is You is a remake of the puzzle game Baba is You, coded in Python. Most controls, including movement, pushing obstacles, and basic pulling of obstacles, are functional. Created for Fundamentals of Programming (6.1010).

2023

Sliding ASCII string on an LED array on a breadboard with an RISC-V microprocessor, coded in C. The sliding text says "poggers" and loops continuously. Created for Introduction to Low-level Programming in C and Assembly (6.1910).
